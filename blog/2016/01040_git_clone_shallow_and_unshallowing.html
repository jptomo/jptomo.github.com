いったん shallow に git clone して、clone 後に完全なリポジトリに変換する¶

2016/01/04

git fetch のオプション --unshallow を利用する。

$ git clone --depth 1 git://github.com/jptomo/acme.hello.git
$ cd acme.hello
$ git fetch --unshallow
